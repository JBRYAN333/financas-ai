# Política de Privacidade — Financas AI

Última atualização: Julho de 2026

## 1. Dados Coletados

Ao utilizar o Financas AI, coletamos:
- **Email e nome:** via login do Google (autenticação)
- **Transações financeiras:** valores, descrições, categorias e datas que você insere voluntariamente
- **Dados de uso:** estatísticas anônimas de funcionalidades acessadas

## 2. Armazenamento

Seus dados são armazenados no **Google Firebase Firestore**, servidores localizados nos Estados Unidos e América do Sul, com criptografia em trânsito (TLS) e em repouso.

Nós **não compartilhamos** seus dados com terceiros.

## 3. Uso da IA

As descrições das suas transações podem ser enviadas para nossa API de inteligência artificial (9Router) exclusivamente para fins de classificação e categorização. Não armazenamos o conteúdo das requisições.

## 4. Seus Direitos (LGPD)

Você tem direito a:
- **Acessar** todos os dados que armazenamos sobre você
- **Corrigir** dados incorretos
- **Excluir** permanentemente sua conta e todos os dados associados
- **Revogar** o consentimento a qualquer momento

Para exercer qualquer direito, entre em contato ou use o botão "Excluir meus dados" no dashboard.

## 5. Retenção

Seus dados são mantidos enquanto sua conta estiver ativa. Após exclusão da conta, todos os dados são removidos em até 30 dias.

## 6. Segurança

- Autenticação via Google OAuth 2.0
- Regras de segurança no Firestore: cada usuário vê **apenas os próprios dados**
- Token de API com acesso restrito a repositórios específicos
